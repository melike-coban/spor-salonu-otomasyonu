# Spor Salonu Otomasyonu

Bu proje Java ile geliştirilmiş basit bir spor salonu otomasyonudur.

## Özellikler
- Üye kayıt sistemi
- MySQL veritabanı kullanımı
- BMI (Vücut Durumu) hesaplama
- Beslenme önerileri (protein / karbonhidrat / yağ)
- Antrenör seçimi
- Swing GUI arayüz

## Kullanılan Teknolojiler
- Java
- Swing
- MySQL
